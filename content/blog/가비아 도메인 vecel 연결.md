# 가비아 도메인 vercel 연결

1. 가비아에서 도메인 주소를 구매
2. vercel 나의 프로젝트의 `domain` 으로 들어가서 내가 구매한 도메인 주소 추가
   ![](https://velog.velcdn.com/images/jbinyim12/post/5305edd2-a450-4878-87c8-c4b9ddd02bc6/image.png)
3. 가비아에서 내가 구매한 도메인 관리 > DNS 정보 > DNS 관리 > 설정 > 레코드 수정 에 추가하면 된다.
   ![](https://velog.velcdn.com/images/jbinyim12/post/19190a6d-b134-4099-bf25-b81861834619/image.png)
   레코드에는 vecel에서 준 레코드를 똑같이 입력하면 된다.
   ![](https://velog.velcdn.com/images/jbinyim12/post/b50d8b53-f90c-4aa9-9173-e81c825f108b/image.png)

이후 저장하면 vecel에서 정보가 등록되면 완료.
